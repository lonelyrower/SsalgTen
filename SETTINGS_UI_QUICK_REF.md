# 系统配置 UI 优化 - 快速参考

## ✅ 三大改进

### 1. 布局：宽条 → 网格
- **之前：** 单列宽条，每行占满屏幕
- **现在：** 响应式网格（1/2/3列）
- **效果：** 单屏显示配置数 **3倍 ↑**

### 2. 细节：强制 → 按需
- **之前：** `system.maintenance_mode` 等技术名称直接显示
- **现在：** "维护模式" + `?` 帮助按钮
- **效果：** 悬停查看详情，界面更清爽

### 3. 顺序：重要性 → 使用频率
- **之前：** 系统 > 监控 > 诊断 > ... > 地图
- **现在：** 地图 > 监控 > 诊断 > 系统 > ...
- **效果：** 常用配置在顶部，无需滚动
